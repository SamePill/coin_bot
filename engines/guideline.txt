업비트의 최소 주문 금액은 5,000원이나, 수수료와 잔돈 청소(Dust Clean) 로직 안정성을 고려해 최소 1회 주문 단위를 
10,000원 이상으로 가정하여 분석했습니다.


1. 🛡️ CORE 엔진 (추세/돌파 매매)
추세를 타고 한 번에 크게 먹는(목표 3.5~5% 이상) 스나이퍼형 엔진. 물타기(피라미딩) 없이 1회 단판 승부.

슬롯 수 가이드
최소 1개 / 적정 1~2개 / 최대 3개
분석: 시장에 동시다발적으로 강력한 돌파(거래량 스파이크 + ADX 25 이상)가 일어나는 경우는 드뭅니다. 
슬롯을 무의미하게 늘리기보다 확실한 놈에게 집중하는 것이 좋습니다.
배정 금액 가이드 (1슬롯 기준)
최소: 3만 원 ~ 5만 원
적정: 전체 예산의 10~20%
분석: 물타기를 하지 않기 때문에 한 번 들어갈 때 시드를 묵직하게 넣어야 수익금이 의미 있게 찍힙니다. 
단, 휩쏘(가짜 돌파)로 인한 본절/손절 시 타격이 있으므로 몰빵은 금물입니다.


2. 🏹 HUNTER 엔진 (낙폭 과대/반등 매매)
투매가 나올 때 과매도(RSI 반등) 및 VWAP 지지선에서 받아먹는 단기 쇼부(45분 타임컷) 엔진. 물타기 없음.

슬롯 수 가이드
최소 1개 / 적정 2~3개 / 최대 4개
분석: 비트코인이 순간적으로 쏟아질 때(패닉장 직전) 여러 알트코인이 동시에 낙폭 과대 자리에 들어올 수 있습니다. 
이때 여러 개를 낚아채기 위해 슬롯이 2~3개 정도 열려있는 것이 유리합니다.
배정 금액 가이드 (1슬롯 기준)
최소: 2만 원 ~ 3만 원
적정: 전체 예산의 10~15%
분석: 떨어지는 칼날을 잡는 셈이라 구조적 손절(직전 저점 이탈)이 자주 발생할 수 있습니다. CORE보다는 비중을 
약간 줄여서 리스크를 관리하는 것이 좋습니다.


3. 🎰 GRID 엔진 (스마트 다이내믹 그리드)
횡보/변동장 코인에 들어가 상승 시 불타기, 하락 시 가중치 물타기(최대 5~8차 이상)를 진행하는 고기동성 엔진.

슬롯 수 가이드
최소 1개 / 적정 2개 / 최대 4개 (단일 코인 중복 슬롯 포함)
분석: 1개 코인당 여러 번의 물타기 로직이 돌아가며 자금을 크게 빨아들입니다. 슬롯이 너무 많으면 WAF(업비트 방화벽) 
API 호출 초과에 걸리기 쉽습니다.
배정 금액 가이드 (1슬롯당 총 할당 예산)
최소: 10만 원 (1회 진입금 1만 원 기준, 최소 5차 이상 방어 대비)
적정: 전체 예산의 30~40%
분석: 하락 시 가중치(get_pyramiding_weight)를 두어 2배, 4배, 6배로 물을 타야 하므로 단일 엔진 중 
가장 예산을 넉넉하게 배정해야 봇이 멈추지 않고 평단가를 방어할 수 있습니다. GRID_UNIT_SIZES 첫 단위의 
최소 10~20배를 슬롯 예산으로 잡아야 합니다.


4. ⚡ SCALP 엔진 (초단타 스캘핑)
짧은 수익(0.6~1.5%)으로 극강의 회전율을 노리는 단타. 하락 시 딱 1번(2차)만 물을 탐.

슬롯 수 가이드
최소 2개 / 적정 3~5개 / 최대 8개
분석: 물리는 시간이 4시간 미만(타임컷)으로 짧고 회전이 빠르므로 다수의 슬롯을 뿌려두고 걸리는 것을 계속 먹고 
나오는 '통발' 전략이 적합합니다.
배정 금액 가이드 (1슬롯당 총 할당 예산)
최소: 4만 원 (1차 1만 원 + 2차 물타기 대비 3만 원)
적정: 전체 예산의 15~20%
분석: 수익률이 1% 내외이므로 진입 시드가 작으면 수수료(0.1%)를 떼고 남는 수익금이 몇 십 원 단위(더스트)가 될 수 
있습니다. 1차 진입 단위(SCALP_UNIT_SIZES)를 최소 2만 원 이상으로 잡는 것을 권장합니다.


5. 🕸️ CLASSIC_GRID 엔진 (전통적 거미줄 매매)
초기 진입 시 50% 빡! 넣고, 나머지 50%를 15%씩 분할 매수/매도하며 박스권에서 평단은 낮추고 코인 개수는 늘리는 
우보마적(牛步馬跡) 엔진.

슬롯 수 가이드
최소 1개 / 적정 1~2개 / 최대 3개
분석: 비트코인, 이더리움, 솔라나 같은 대장주나 확실한 메이저 알트코인에 1코인 1슬롯으로 우직하게 박아두는 용도입니다.
배정 금액 가이드 (1슬롯당 총 할당 예산)
최소: 15만 원 ~ 20만 원
적정: 전체 예산의 20~30%
분석: 최초에 예산의 50%가 투입되고, 나머지 예비비로 15%씩 부분 매수/매도를 진행합니다. 매수/매도 단위가 최소 
주문 금액(6,000원)을 넘기려면 전체 할당 예산이 꽤 커야 합니다. (예비비 7.5만원의 15% = 약 1.1만원 주문).
💡 총 시드별 포트폴리오(엔진 활성화) 추천 가이드
🥉 총 예산 50만 원 미만 (소액/테스트)
자본이 적을 때는 모든 엔진을 켜면 주문 금액 미달(Dust) 에러가 발생합니다. 핵심 엔진 1~2개만 가동하세요.

추천 엔진: SCALP (60%) + CORE (40%)
세팅: SCALP 슬롯 2~3개 (회전율로 소소한 수익 창출), CORE 슬롯 1개 (단기 추세 돌파로 큰 수익 노림). 
GRID류는 물타기 자금 부족으로 위험합니다.
🥈 총 예산 100만 원 ~ 300만 원 (일반 운영)
가장 보편적인 바벨(안정+공격) 전략 포트폴리오를 구성할 수 있습니다.

추천 엔진: GRID (40%) + CLASSIC_GRID (30%) + SCALP (20%) + CORE/HUNTER (10%)
세팅: 변동성을 발라먹는 GRID 2슬롯에 예산을 가장 많이 주고, 든든한 대장주 1개를 CLASSIC_GRID에 배치합니다. 
남는 예산으로 SCALP(3슬롯)와 단발성 CORE를 운용합니다.
🥇 총 예산 500만 원 이상 (고액 운영)
자금의 여유가 생기면 공격적인 스캘핑이나 돌파보다는 시장의 파동을 다 받아내는 그리드와 낙폭과대에 비중을 싣습니다.

추천 엔진: CLASSIC_GRID (40%) + GRID (30%) + HUNTER (20%) + CORE (10%)
세팅: 우량주 2~3개를 CLASSIC_GRID에 거미줄 쳐놓고, 횡보/변동장 알트코인 3개 정도를 스마트 GRID로 돌립니다. 
100만 원 정도를 HUNTER에 할당하여 하락장/투매 시 저점 줍기용 저격수로 활용합니다. SCALP는 관리 리소스 대비 피로도가 높으므로 비중을 낮추거나 끕니다.
